Welcome

This online CV is an experiment and learning experience in data visualization, using the Shiny app for R and infographics.

A CV is just a collection of data, typically presented in a relatively standard (and somewhat boring) format. The Shiny app is designed to let users interact with data in ways that are interesting, informative, and fun. Although this CV isn't exactly "big data," the Shiny app still makes parsing through the information more interesting.

Please take a look around, and, if you're interested, you can freely browse the code on GitHub

About me

I am a veterinarian who recently completed my PhD in the immunogenetics of the innate immune system of livestock. I have an interest in using big data, especially in the context of One Health, to improve the health of animals and humans.
