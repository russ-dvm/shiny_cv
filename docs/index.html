Welcome

This online CV is an experiment and learning experience using the Shiny app for R.

Although typically used for interactive visualization of traditional datasets, a CV is really just a collection of data. The Shiny app is designed to let users interact with data in ways that are interesting, informative, and fun. This CV isn't exactly "big data," but the Shiny app still makes parsing through a CV somewhat more interesting.

Please take a look around, and, if you're interested, you can freely browse the code on GitHub
